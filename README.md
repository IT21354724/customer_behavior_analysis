#Customer Shopping Behavior Analysis

This project analyzes customer shopping behavior using transactional data from 3,900 purchases across multiple product categories. The objective is to identify spending patterns, customer segments, product preferences, and subscription behavior to support data-driven business decisions.

The workflow includes data cleaning and feature engineering using Python, structured business analysis using PostgreSQL, and interactive dashboard creation using Power BI. Key insights such as revenue trends, customer segmentation, discount usage, and high-performing products are visualized to provide actionable business recommendations.

#Tools & Technologies:
Python (Pandas, NumPy), PostgreSQL (SQL), Power BI
